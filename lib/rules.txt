! Imp Translate - Site-specific rules
! Only targets the world's top 50 most-visited websites.
! Reference: https://en.wikipedia.org/wiki/List_of_most-visited_websites
!
! Syntax:
!   domain##selector   — skip (do not translate)
!   domain#+#selector  — include (only translate inside)
!   entity.* — match any TLD (e.g. google.* matches google.com, google.com.hk, google.co.uk)
!
! Each rule (or rule group) is preceded by a `!` URL comment listing the
! page(s) it was verified against. This lets future maintainers re-visit
! the page, confirm whether the rule is still needed, and safely remove
! obsolete rules — without URL annotations the list is a one-way ratchet.

! Google Search (www.google.* avoids matching mail/drive/docs subdomains)
! https://www.google.com/search?q=test
www.google.*#+#[id="rcnt"]
www.google.*##:has(>[hreftranslate])
www.google.*##[role="navigation"]

! Reddit — component-based platform; include the post/comment text slots only. Author names, vote counts, action rows, and chrome (header, sidebar) sit outside the include set automatically.
! https://www.reddit.com/r/programming/  (subreddit feed: titles + text-body for text posts)
! https://www.reddit.com/r/programming/comments/<id>/  (post detail: title + body + comments)
reddit.com#+#[slot="title"]
reddit.com#+#[slot="text-body"]
reddit.com#+#[slot="comment"]
! Reddit Chat — message bodies + link-preview cards live inside nested Shadow DOM (rs-timeline-event > shadow > rs-external-link-preview > shadow). The walker pierces open shadow roots, and include matching crosses shadow boundaries via closestThroughShadow.
! https://chat.reddit.com/  (any chat room — actual room URLs are private)
reddit.com#+#.room-message-text
reddit.com#+#rs-external-link-preview

! Wikipedia — document-style site; default walker handles main content. Only one nuisance: section-heading [edit] links bleed into the heading translation.
! https://en.wikipedia.org/wiki/Public_Suffix_List
wikipedia.org##.mw-editsection

! GitHub — document-style site; default walker handles README / issue / PR body via .markdown-body. Skip the repo home file tree where filenames merge with commit messages.
! https://github.com/microsoft/vscode  (repo home: file tree)
github.com##.react-directory-row

! ChatGPT — chat platform; include user prompts and assistant responses only. Code blocks (<pre>) are already skipped by the default walker.
! https://chatgpt.com/c/<id>  (any conversation)
chatgpt.com#+#[data-message-author-role]

! Discord — chat platform; include message text and accessories. Reply-context preview is covered automatically because Discord renders the quoted text as another [id^="message-content-"] inside the reply block. Channel sidebar, member list, and the message composer fall outside the include set.

discord.com#+#[id^="message-content-"]
